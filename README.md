# S. Aromal - Personal Portfolio

This is the personal portfolio and resume website of S. Aromal, Founder at Syncify Edutech Solutions.

## Tech Stack
- **Framework:** [Astro](https://astro.build)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com)
- **Interactivity:** [Alpine.js](https://alpinejs.dev)

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run the development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```

The static site will be generated in the `dist/` directory, ready to be deployed.
